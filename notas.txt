Instalar proyecto
sudo git  clone https://github.com/danizd/dominios --branch master
pip install -r requirements.txt

Crear Entorno virtual dentro de la carpeta del proyecto
python3 -m venv myvenv
source myvenv/bin/activate
pip install -r requirements.txt

Iniciar servidor
python3 manage.py runserver

Subir a github
git status
git add --all .
git status
git commit -m "Cambie el HTML para la página."
git push

Descarga tu nuevo código a PythonAnywhere y actualiza tu aplicación web

cd ~/puntogal.pythonanywhere.com
git pull
